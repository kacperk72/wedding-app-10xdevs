# Harmonogram wesela (ankieta DJ-a)

Nowa zakładka w wedding-plannerze, w której para wypełnia ankietę otrzymaną od DJ-a:
godzinowy przebieg dnia, pytania logistyczne, preferencje muzyczne i listy utworów
(must-play / do-not-play), a następnie eksportuje „Wersję dla DJ-a" do druku/PDF.

Reuse istniejących danych wesela (imiona pary, miejsce ceremonii, liczba gości, DJ/sala
z kontrahentów) zamiast ich ponownego zbierania.
